# Portfolio

https://jeeannyy.github.io/Jeeann

## 🛠 set-up

1. Install the dependencies

   ```
   npm install
   ```

2. Start the development server

   ```
   npm start
   ```

## 🚀 build and run for production

1. Generate a full static production build

   ```sh
   npm run build
   ```
